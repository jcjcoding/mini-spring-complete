## 目录结构说明
- `JavaBackend/`：按照章节记录学习情况（按主题分目录（如`reflection/`反射、`collection/`集合）
- `algorithm/daily/`：每日LeetCode题解（按日期命名）
- `algorithm/topic/`：按题型分类的算法题解（如数组、链表）
- `docs/`：项目文档（计划概述、学习笔记、架构图等）
